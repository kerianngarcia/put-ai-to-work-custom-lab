# Section 3. Now Assist for the Agent Persona

In this section, you will explore how Now Assist helps agents work more efficiently inside Service Operations Workspace.

You will summarize incidents and change requests, generate resolution notes, create knowledge articles, and use AI Agents to identify configuration items impacted by change requests.

{% hint style="info" %}

### In this section, you will

☐ Generate incident summaries with Now Assist.

☐ Create and publish incident resolution notes.

☐ Generate knowledge articles from resolved incidents.

☐ Summarize change requests.

☐ Use an AI Agent to identify impacted configuration items.

{% endhint %}
