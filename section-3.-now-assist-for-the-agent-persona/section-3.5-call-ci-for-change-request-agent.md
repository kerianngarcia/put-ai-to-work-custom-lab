# Section 3.5 Call 'CI for Change request' agent
1. From the same change request, click on the Now Assist panel sparkle in the top right-hand corner
![](/files/Ifd3vHXzf2uzoDCRbjiv)

2. This will expand all the Skills and Agents that this user has permission to use.
![](/files/a90U4cO5ZbmcvJieKK7C)

2. Click on the Suggest "\*\*Configuration Items for a change request\*\*” card.
![](/files/OHiIQtVTso0Y0JO7rXss)

2. This agent searches the CMDB for CIs that may be impacted by this change.
You could select any of the Cis (or multiple) to associate with the change
\*\*Congratulations!\*\* You have summarized and identified Cis impacted by a change request and completed Now Assist for the Agent persona.
